Hola, esta es una guia de GIT

Pasos para ejecutar GIT

Todos los pasos se ejecutan desde la terminal o cd

  1. GIT version: comprueba que version tenemos
  2. GIT Add: para agregar archivos
  3. GIT Commit: los etiqueta
  4. GIT Push: los empuja a la nube
  5. GIT help: pedir ayuda
  6. Limpiar la consola: cls

Pasos desde la terminal

  1. git snit: iniciar repositorio
  2. git status -s: se ve lo que esta pasando
  3. git config --global user.mail "email": para revisarlo escribir git config user.name
  4. git config --global user.name "name": para revisarlo escribir git config user.email
  5. git add . : copia todos los archivos que tengamos
  6. git add index.html : copia solamente la carpeta que escribamos
  7. git commit -m "mensaje" : etiquetar el repositorio que contiene esas carpetas. Si escribimos solo git commit+enter: aparece un aviso y para cerrarlo escribir q! o :q!
  8. git log --oneline : para ver que nombre le pusimos

En la carpeta de Windows/Linux donde alojemos el proyecto se creara una carpeta oculta para verla apretar en Vista - Elementos Ocultos

Pasos desde los botones de VSC

  1. Ir Source Control-Initialize Repository: los coloca en U
  2. Apretar en + seria git add
  3. En mensaje escribir el nombre del archivo que seria git commit

Como subirlo a la nube?

  1. Ir al sitio web GITHUB
  2. Ir a + New Repository
  3. Escribir el nombre de la carpeta, elegir publico o privado, aceptar
  4. copiar remote add origen, branch y push y pegarlo en la terminal por separado
  5. copiar branch -M master y pegarlo en la terminal
  6. copiar push y pegarlo en la terminal

Si queremos agregar mas info

  1. Guardarla
  2. Escribir en la terminal git add .
  3. Escribir en la terminal git commit -m "nombre"
  4. Escribir git push y lo envia

Como utilizar Github page?

  1. Desde la pagina github ir a setting - Pages - elegir la rama master
  2. Esperar a que se cargue toda la pagina, demora un tiempito
